Вопросы

Часто спрашивают

С чего начать работу с Aurora? +
Просто зарегистрируйтесь и получите доступ к бесплатному плану. Никаких кредитных карт, никаких обязательств — просто начните создавать.
Можно ли отменить подписку в любое время? +
Конечно. Вы можете отменить подписку в любой момент в настройках аккаунта. Никаких скрытых условий и штрафов.
Как обстоит дело с безопасностью данных? +
Мы используем end-to-end шифрование, регулярно проходим аудиты безопасности и соответствуем стандартам SOC 2 Type II и GDPR.
Есть ли API для интеграции? +
Да, мы предоставляем полноценный REST API и WebSocket для real-time подключений. SDK доступны для всех популярных языков.
Поддерживается ли командная работа? +
Безусловно. Aurora изначально создавалась для команд: совместное редактирование, права доступа, история изменений и многое другое.
